# Challenge 15: View File Start and End

## 🧩 Task
Display first and last lines of a file.

---

<details>
<summary>💡 Click to view solution</summary>

### 🔹 View beginning of file
```bash
head file.txt
```

**Example input (`file.txt`):**
```text
line1
line2
line3
line4
line5
line6
```

**Output (head):**
```text
line1
line2
line3
line4
line5
```

---

### 🔹 View end of file
```bash
tail file.txt
```

**Output (tail):**
```text
line2
line3
line4
line5
line6
```
