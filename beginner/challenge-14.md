# Challenge 14: Sort File Content

## 🧩 Task
Sort lines in a file.

---

<details>
<summary>💡 Click to view solution</summary>

### 🔹 Sort file content (default)
```bash
sort file.txt
```

**Example input (`file.txt`):**
```text
banana
apple
orange
grape
```

**Output:**
```text
apple
banana
grape
orange
```

---

### 🔹 Reverse sort
```bash
sort -r file.txt
```

**Output:**
```text
orange
grape
banana
apple
```

---

### 🔹 Numeric sort
```bash
sort -n numbers.txt
```

**Example input (`numbers.txt`):**
```text
10
2
33
5
```

**Output:**
```text
2
5
10
33
```
