# Challenge 29: Use Manual Pages

## 🧩 Task
View documentation for a command.

---

<details>
<summary>💡 Click to view solution</summary>

### 🔹 View manual page for a command
```bash
man ls
```

### 🔹 Example navigation inside man page
- Press **Space** → scroll down  
- Press **b** → scroll up  
- Press **q** → quit  

### 📖 Explanation
- `man` → Opens the manual (documentation) for a command  
- Provides detailed information including:
  - Description of the command  
  - Available options/flags  
  - Usage examples  
  - Syntax and arguments  

**Tip:**
You can search within a man page by typing:
```bash
/keyword
```

Example:
```text
/search
```
