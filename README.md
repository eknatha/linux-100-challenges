# Linux 100 Challenges for DevOps / SRE / System Administrators

A curated collection of **100 hands-on Linux challenges** designed to build real-world skills for:

- DevOps Engineers
- Site Reliability Engineers (SRE)
- System Administrators
- Cloud Engineers

---

## 📌 Overview

This repository contains practical Linux challenges from **beginner to advanced level**, focused on:

- Linux fundamentals
- File system & permissions
- Process management
- Networking
- Security
- Monitoring & troubleshooting

---

## 🧠 Difficulty Levels

- 🟢 Beginner (1–30)
- 🟡 Intermediate (31–70)
- 🔴 Advanced (71–100)

## 📝 Challenge Format

Each challenge can include:

- Problem statement
- Expected outcome
- Commands used
- Optional solution

Example:

```
### Challenge 1: List Hidden Files

Task: List all files including hidden files.

Solution:
ls -la
```
